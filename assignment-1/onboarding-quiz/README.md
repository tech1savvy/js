# Interactive Onboarding Quiz App

This project is a simple, interactive quiz application built with HTML, CSS, and JavaScript. It is designed to test new employees on company values and policies.

## Project Description and Purpose

This application presents a 5-question multiple-choice quiz. It provides immediate feedback to the user after each question and displays a final score and a performance-based message at the end. The user can retake the quiz as many times as they like.

The purpose of this project is to demonstrate fundamental knowledge of front-end web development, including DOM manipulation, event handling, and state management in JavaScript.

## JS Concepts Used

*   **Variables:** `let` and `const`
*   **Data Types:** `string`, `number`, `boolean`, `array`, `object`
*   **Control Flow:** `if/else`, `for/forEach`
*   **Functions:** Function declarations and arrow functions
*   **DOM Manipulation:**
    *   `document.getElementById`
    *   `document.querySelector`
    *   `document.createElement`
    *   `element.textContent`
    *   `element.innerHTML`
    *   `element.classList`
    *   `element.addEventListener`
*   **Asynchronous JavaScript:** `setTimeout`

## Instructions to Run

1.  Clone this repository or download the files.
2.  Open the `index.html` file in your web browser.

That's it! The quiz will be ready to use.
